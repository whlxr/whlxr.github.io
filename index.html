Markdown从精通到入门

1. 字体

斜体用一个*号包围内容
这是斜体，内容两端被一个星号包围
粗体用两个*号包围是内容
这是粗体，内容两端被两个星号包围

2. 列表

  • 无序列表用减号，如-，一个空格，再加内容
  • 在标识符
  • 后面要有一个空格
  1. 有序列表用数字+英文句号，如1.，后面加空格再加内容
  2. 也一样
  3. 要有空格

3. 超链接

外链接用中括号描述和小括号链接,格式为[描述](URL)

这是百度

4. 段落引用

右箭头>表示文字引用，不同解析器会有不同

这是引用的文这。在一行文字的结尾敲两个空格，再敲一个回车，就会强制换行
这是换行的文字

这是第三行

加粗，有标记的换行不需要加两个空格

5. 代码块

用`(反单引号，在ESC键上,不是单引号)包住代码，表示行内代码块html css javascript Python

普通代码块的表示，使用四个缩进空格 表示代码块。

public static void main(String[] args){
    System.out.println("I'm your father");
}

加强的代码块用ESC键的那个，三个反单引号后面加上语言，比如java，Python，如```java,代码结尾用三个返单引号结束

插入图像和插入链接一样，只不过在前面加个！
